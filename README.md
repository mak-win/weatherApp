# Weather App

A simple web application that lets users search for current weather information for any city using the OpenWeatherMap API.

## Features

- Search for a city by name
- Display current temperature, humidity, and weather description
- Handles errors for invalid city names
- Loading indicator while fetching data
- Displays the latest search result only

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
